Personal Finance Mobile App Overview

A mobile financial management application designed for expense tracking, banking analysis, synchronization with the desktop platform, and real-time financial monitoring.

Desktop & Mobile Synchronization

Synchronization interface connecting the mobile application with the desktop financial management software.

Main Mobile Application Interface

Main dashboard of the mobile app providing access to financial tools, expense tracking, and banking analytics.

Monthly Expense Comparison Spinner

Spinner displaying current monthly expenses for the selected date compared with the same point of the previous month.

Bank Transaction Registration

Main interface for recording and managing bank transactions directly from the mobile application.

Bank Account Balance Spinner

Spinner displaying bank accounts, current balances, and balance trends compared with the same day of the previous month.

Monthly Account Balance Analysis

Chart displaying monthly balances for the selected bank account or the combined total of all accounts during the selected year.

Annual Account Balance Distribution

Donut chart visualizing final annual balances distributed across all bank accounts.

Expense Radar Analysis

Radar chart displaying total monthly expenses or annual expenses for the selected year.

Annual Expense Distribution

Donut chart visualizing annual spending distribution for the selected year.
